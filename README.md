# Newsletter Subscription Card

A clean and modern newsletter subscription card built using **Tailwind CSS** and **HTML5**.

This project was created to practice and demonstrate utility-first styling with Tailwind, including layouts, gradients, spacing, typography, and component design.

## Features

* Utility-first styling using Tailwind CSS
* Gradient header section
* Responsive Flexbox layout
* Styled form input and CTA button
* Rounded cards and modern UI design
* No custom CSS required for component styling

## Installation

Clone the repository:

```bash
git clone https://github.com/TheNidhishGarg/tailwind-newsletter-card
```

Install dependencies:

```bash
npm install
```

Start Tailwind in watch mode:

```bash
npm run dev
```

Open `index.html` in your browser.

---

## Build for Production

Generate an optimized production stylesheet:

```bash
npm run build
```
